<h1>🛍️ COSTY Fashion Website</h1>

<p>
This project is a <strong>modern fashion e-commerce landing page</strong> created for the fictional clothing brand <strong>COSTY</strong>. 
It focuses on a stylish and minimal design while showcasing fashion products in an attractive way.
</p>

<p>
The website is built using <strong>HTML</strong>, <strong>CSS</strong>, and <strong>JavaScript</strong>. 
It includes responsive design, smooth animations, and dynamic product rendering to create a modern shopping experience.
</p>

<h1>✨ Features</h1>

<p><strong>Responsive Design:</strong> Works smoothly on desktop, tablet, and mobile devices.</p>
<p><strong>Modern UI:</strong> Clean and minimal fashion brand inspired layout.</p>
<p><strong>Hero Section:</strong> Eye-catching landing area highlighting the COSTY brand.</p>
<p><strong>Dynamic Product Listing:</strong> Products are displayed dynamically using JavaScript.</p>
<p><strong>Shopping Cart Icon:</strong> Basket indicator showing the number of selected items.</p>
<p><strong>Smooth Animations:</strong> Scroll animations implemented with ScrollReveal.</p>
<p><strong>Structured Footer:</strong> Includes social media links, help section, resources, and company information.</p>

<h1>📚 Technologies Used</h1>

<p>HTML5</p>
<p>CSS3</p>
<p>JavaScript (ES6 Modules)</p>
<p>Bootstrap Icons</p>
<p>Remix Icons</p>
<p>Google Fonts (Montserrat)</p>
<p>ScrollReveal.js</p>
